CSS Inheritance

Inherited:

font-size, font-family, font-style, font-weight, color, text-align, text-transform, text-indent, line-height, letter-spacing, word-spacing, white-space, direction and etc.

Not inherited:

background, border, padding, margin, width, height, position and etc

Efficiently Rendering CSS

            header nav ul li a !important
            header > nav > ul > li > a
            ul.nav a.nav-link
            .nav-link

You should avoid

  • Inline styles
  • !important
  • ID selectors (and tag selectors, as much as possible)
  • Huge selectors or cascades

Block

display: block;
  • Inline formatting contexts
  • Adjustable width, height
  • Take all horizontal available place

Inline

display: inline;
  • Inline formatting contexts
  • Cannot set width, height
  • Vertical margin and padding do not affect elements position

Inline-Block

display: inline-block;
  • Block formatting contexts
  • Adjustable width, height, margin, padding
  • Behaves like a symbol
