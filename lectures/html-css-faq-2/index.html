You should avoid

  • Inline styles
  • !important
  • ID selectors (and tag selectors, as much as possible)
  • Huge selectors or cascades

Block

display: block;
  • Block formatting contexts
  • Adjustable width, height
  • Take all horizontal available place

Inline

display: inline;
  • Inline formatting contexts
  • Cannot set width, height
  • Vertical margin and padding do not affect elements position

Inline-Block

display: inline-block;
  • Inline formatting contexts
  • Adjustable width, height, margin, padding
  • Behaves like a symbol

CSS Units

Relative Lengths

Unit relative to
% to the same property of the parent element
em font size of the element
rem font size of the root element
vw 1% of viewport’s width
vh 1% of viewport’s height
vmax 1% of viewport’s smaller dimension
vmin 1% of viewport’s larger dimension

Absolute Lengths

Unit equivalence
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)

Notes

  • em and px are most usable units for fonts, rem becoming popular.
  • % is the most popular unit for layout due to responsive.
  • vw/vh will become popular with new Grid Layout.
  • Most of absolute units are used preferably for print page rules.

Learn more here: W3C

Notes

  • Element with [positive] z-index always will be above element without it.
  • Lower amount of z-index elements is better.
  • On the same level the last element will be the highest one.
  • In most cases you actually do not need z-index.
