2D Трансформации

transform-origin – точка опоры

transform-origin: x y z
  • x длина | проценты | left | center | right
  • y длина | проценты | top | center | bottom
  • z длина

Transform

  • translate - сдвиг
  • scale - масштабирование
  • rotate - поворот
  • skew - наклон
  • matrix - матрица

Translate – сдвиг

transform: translate(x[, y])
            translateX(x)
            translateY(y)

Код:

transform: translateX(50px) translateY(-20px);
transform: translate(50px, -20px);

Scale – масштабирование

transform: scale(x[, y])
            scaleX(x)
            scaleY(y)

Код:

transform: scale(2);
transform: scale(2,1)

Rotate – поворот

transform: rotate(угол)

Код:

transform: rotate(180deg);

Skew – наклон

transform: skew(x-угол,y-угол)
            skewX(угол)
            skewY(угол)

Код:

transform: skew(35deg);
transform: skew(30deg, 20deg);

Matrix – матрица

transform: matrix(a, c, b, d, tx, ty);

        a   b   0
        c   d   0
        tx  ty  1

Коэффициенты матрицы преобразований

  • a – изменение масштаба по горизонтали.
  • b – наклон по горизонтали
  • c – наклон по вертикали
  • d – изменение масштаба по вертикали
  • tx – смещение по горизонтали в пикселах
  • ty – смещение по вертикали в пикселах
