DRY CSS

by Jeremy Clarke

DRY: DON’T REPEAT YOURSELF

transform: translate(x[, y])
            translateX(x)
            translateY(y)

Код:

transform: translateX(70px) 
            translateY(-30px);
transform: translate(70px, -30px);

Scale – масштабирование

transform: scale(x[, y])
            scaleX(x)
            scaleY(y)

Код:

transform: scale(1,2);
transform: scale(2,1)

Rotate – поворот

transform: rotate(угол)

Код:

transform: rotate(45deg);

Skew – наклон

transform: skew(x-угол,y-угол)
            skewX(угол)
            skewY(угол)

Код:

transform: skew(35deg);
transform: skew(-30deg, -20deg);

Matrix – матрица

transform: matrix(a, b, c, d, tx, ty);

        a   b   0
        c   d   0
        tx  ty  1

Коэффициенты матрицы преобразований

  • a – изменение масштаба по горизонтали.
  • b – наклон по горизонтали
  • c – наклон по вертикали
  • d – изменение масштаба по вертикали
  • tx – смещение по горизонтали в пикселах
  • ty – смещение по вертикали в пикселах

3D Трансформации

Transform-style

transform-style: flat | preserve-3d;
flat – дочерние элементы лежат в той же плоскости, что и их родитель
preserve-3d – дочерние элементы будут отображаться в 3D-пространстве.

Перспектива

perspective – как сильно изменится блок при трансформации
css-tricks.com/perspective

css-tricks.com/perspective-origin

3D Transform

  • translate3D - сдвиг
  • rotate3D - поворот
  • scale3D - масштабирование
  • matrix3D - матрица

Translate3D – сдвиг

transform3D: translate3D(x,y,z)
              translateX(x)
              translateY(y)
              translateY(z)

Код:

transform: 
        translate3D(70px, -30px, 50px);

Rotate3D – поворот

transform: rotate3D(x,y,z,угол)

Код:

transform: rotate3D(1, -1, 1, 45deg);

Scale3D – масштабирование

transform: scale3D(x,y,z)
                scaleX(x)
                scaleY(y)
                scaleY(z)

Код:

transform: scale3D(1, 0.5, 1) rotateY(-20deg);
transform: scale3D(1, 0.5, 3) rotateY(-20deg);
