При помощи CSS3 можно:

  • скруглять,
  • вращать,
  • перемещать
  • и даже анимировать
  • в ЗD!!!

Поговорим
о кругах

border-radius – скругляем уголки.

border-radius: a b c d;
a b d c

Можно так:

border-radius: 100px; height: 200px;
width: 300px;
border-radius: 50%; height: 200px;
width: 300px;

Или так:

border-radius: 100% 0;
border-radius: 13% 25% 38% 50%;

В жизни:

иконки
фото в круге

box-shadow – тень для блока.

box-shadow: сдвиг по x
			 сдвиг по y
			 радиус размытия
			 растяжение
			 цвет
			 inset - если внутренняя

Что получаем:

box-shadow: 10px 10px 30px 0 #ffcc00;

Теней может быть много:

box-shadow: 10px 10px 30px 0 #ff6600,
			 10px 10px 30px 0 #ff6600 inset;

Eще больше:

box-shadow: 0 0 5px 10px #ff0000,
			 0 0 5px 20px #ffcc00,
			 0 0 5px 30px #ffff00,
			 0 0 5px 40px #00ff33,
			 0 0 5px 50px #00ffff,
			 0 0 5px 60px #3300ff,
			 0 0 5px 70px #ff00ff;

text-shadow: сдвиг по x
			  сдвиг по y

			  цвет

Красивые фоны

background-size – масштабируем фон


background-size:значение / проценты / auto / 
				 cover / contain

А вы видите кота???

background-image: url(../pictures/cat.jpg);

А кот есть!

background-image: url(../pictures/cat.jpg);
background-size: 100% auto;

Другие значения:

100% 100%
200px 150px
cover
contain

Линейный градиент – это фоновое изображение.

background-image: linear-gradient
			(направление,
			начальный цвет,
			промежуточные цвета и их положение в %,
			конечный цвет)

Двухцветный градиент:

background-image: linear-gradient(green, blue);

Немного сложнее:

background-image: linear-gradient(to top right, red, white, blue);

Несколько градиентов:

background-image: linear-gradient(to top left, rgba(255,255,0,0), #99ff00),
linear-gradient(to top right, rgba(255,255,0,0), red);

Орнамент из градиентов:

background-color: #026873 no-repeat;
background-image: linear-gradient(to left, transparent 50%, rgba(255,255,255,.5) 50%);
background-size: 50px auto;

Повторяем фрагмент:

background-color: #026873
background-image: linear-gradient(to left, transparent 50%, rgba(255,255,255,.5) 50%);
background-size: 50px auto;

Если градиентов несколько:

background-image: linear-gradient(to right, rgba(255,255,255,.07) 50%, transparent 50%),
linear-gradient(to right, rgba(255,255,255,.13) 50%, transparent 50%),
linear-gradient(to right, transparent 50%, rgba(255,255,255,.17) 50%),
linear-gradient(to right, transparent 50%, rgba(255,255,255,.19) 50%);
background-size: 13px, 29px, 37px, 53px;
